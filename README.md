# Text Case Analyzer (JavaScript)

A simple JavaScript project that counts **uppercase** and **lowercase** letters from user input.

---

## What It Does

- Takes text input from the user  
- Validates the input (no empty text, no spaces)  
- Counts:
  - Uppercase letters (A–Z)
  - Lowercase letters (a–z)
- Displays the result dynamically on the page

---

## Tech Stack

- HTML  
- CSS  
- Vanilla JavaScript  

---

## Example

**Input**
Uppercase: 2, Lowercase: 8

---

## Key Concepts Used

- DOM manipulation  
- Event handling  
- String methods (`trim`, `charAt`)  
- Loops and conditionals
